# CRC-721 Non-Fungible Token Standard

The CRC-721 non-fungible token (NFT) standard defines the basic functionality and interface for unique, non-interchangeable tokens on the ClawChain network. This includes methods for:

- Querying information about a specific token
- Transferring ownership of tokens
- Approving other addresses to manage specific tokens

## Token Metadata

Every CRC-721 token must have the following metadata:

- **Name**: The human-readable name of the token collection
- **Symbol**: The short ticker symbol for the token collection
- **tokenURI(uint256 tokenId)**: A URI pointing to metadata about the specific token

## Required Functions

The CRC-721 standard defines the following required functions:

**balanceOf(address owner)**: Returns the number of NFTs owned by the given address.

**ownerOf(uint256 tokenId)**: Returns the address of the owner of the given NFT.

**safeTransferFrom(address from, address to, uint256 tokenId)**: Safely transfers the ownership of a specific NFT from one address to another.

**transferFrom(address from, address to, uint256 tokenId)**: Transfers the ownership of a specific NFT from one address to another.

**approve(address to, uint256 tokenId)**: Grants the approved address the right to transfer the specified NFT.

**getApproved(uint256 tokenId)**: Returns the address that is approved to transfer the given NFT.

**setApprovalForAll(address operator, bool approved)**: Enables or disables the operator to manage all of the caller's tokens.

**isApprovedForAll(address owner, address operator)**: Returns whether the given operator is approved by the owner to manage all of the owner's NFTs.

## Events

The CRC-721 standard defines the following events:

**Transfer(address indexed from, address indexed to, uint256 indexed tokenId)**: Emitted when ownership of a specific NFT changes.

**Approval(address indexed owner, address indexed approved, uint256 indexed tokenId)**: Emitted when a specific NFT is approved for a given address.

**ApprovalForAll(address indexed owner, address indexed operator, bool approved)**: Emitted when an operator is enabled or disabled for an owner.

## Optional Extensions

The CRC-721 standard also defines several optional extensions that tokens may implement:

- **Enumerable**: Allows for enumeration of all NFTs in the collection.
- **Metadata**: Provides a standard way to access token metadata.

Implementers should refer to the relevant extension specifications for details on these optional features.
